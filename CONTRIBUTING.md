# Kontribusi

Terima kasih sudah mau berkontribusi ke CekLaptop.

## Cara mulai

1. Fork repo ini, lalu clone fork-mu.
2. Semuanya ada di satu file: `index.html` (HTML, CSS, dan JS digabung, tanpa build step).
3. Buka `index.html` langsung di browser untuk lihat perubahan — tidak perlu install dependency apa pun.
4. Buat branch baru, commit perubahanmu, lalu buka Pull Request.

## Ide kontribusi

- Tes hardware baru (mis. fingerprint reader, port tertentu).
- Perbaikan copy/instruksi Bahasa Indonesia yang kurang jelas.
- Perbaikan kompatibilitas browser/OS lain (saat ini fokus ke Windows).
- Terjemahan ke bahasa lain.

## Panduan singkat

- Pertahankan gaya kode yang sudah ada (vanilla JS, tanpa framework/dependency eksternal selain 7.css untuk styling dan Google Fonts).
- Jangan tambahkan build tooling — proyek ini sengaja tetap satu file statis supaya siapa pun bisa host di mana saja (GitHub Pages, dsb) tanpa proses build.
- Untuk perubahan besar, buka issue dulu buat didiskusikan sebelum mulai coding.
