Tes Interaktif

Jalankan tiap tes, lalu tandai hasilnya: Lolos, Gagal, atau Lewati kalau tidak relevan.

Dead Pixel, Backlight Bleed & Banding

Layar penuh berganti pola: 6 warna solid (kuning menguji subpiksel merah+hijau sekaligus), lalu gradasi abu (cari garis-garis bertingkat/banding) dan hitam pekat — di layar hitam, redupkan lampu ruangan dan cari cahaya bocor di tepi layar (backlight bleed). Klik untuk lanjut, Esc keluar.

Keyboard: Tombol Mati, Chattering & Rollover

Tekan tiap tombol — hijau berarti terdeteksi. Tombol merah berarti terdeteksi ganda dalam sekali tekan (chattering, penyakit umum keyboard bekas). Tekan banyak tombol bersamaan untuk mengukur rollover. Catatan: tombol Fn tidak terlihat oleh browser; F1–F12 di sebagian laptop perlu ditekan bersama Fn.

Putar nada per kanal — suara harus keluar dari sisi yang benar, jernih, tidak pecah. Sweep frekuensi membongkar speaker sember di frekuensi tertentu.

Webcam

03

Toolkit Lanjutan

Browser tidak bisa membaca hardware level rendah (SMART disk, wear baterai, suhu). Gunakan perintah bawaan Windows di bawah — salin, lalu tempel di laptop yang dites.

Perintah Bawaan Windows (tanpa install)

Buka Command Prompt: tekan Win+R, ketik cmd, Enter. Lalu tempel perintah dan tekan Enter.

powercfg /batteryreportCMD
Cara baca: buka file battery-report.html yang dibuat. Bandingkan Design Capacity vs Full Charge Capacity. Sisa di bawah 80% = baterai sudah aus, bahan nego. Di bawah 60% = siap-siap biaya ganti baterai.
wmic diskdrive get model,status,sizeCMD
Cara baca: Status harus OK. Sekalian cocokkan model & kapasitas drive dengan klaim iklan (SSD vs HDD kelihatan dari nama modelnya).
dxdiagWin+R
Cara baca: tab System = CPU & RAM asli, tab Display = GPU asli. Cocokkan dengan iklan dan dengan pembacaan GPU di header halaman ini.
slmgr /xprCMD
Cara baca: menampilkan status aktivasi Windows — permanen, sementara, atau tidak aktif. Kalau penjual mengklaim "Windows original", di sinilah buktinya.
wmic bios get serialnumberCMD
Cara baca: nomor seri dari BIOS — harus sama dengan stiker di body dan dus/invoice. Beda = tanda ganti motherboard atau unit tidak sesuai.
Aplikasi Portable (siapkan di flashdisk)

Semua gratis dan punya versi portable — jalan tanpa install. Tombol "Unduh" membuka halaman resmi vendor (bukan link file langsung, karena nomor versi sering berubah) — unduh sekali di rumah, pilih edisi Zip/Portable, simpan di flashdisk.
